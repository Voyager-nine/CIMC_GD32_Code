# oled问题排查

## Goal
排查 `oled_app.c` 中 OLED 显示异常问题，定位根因并给出修复方向。

## Requirements
- 检查 OLED 显示字符发生翻转的原因。
- 检查显示字符串不完整（截断/缺字/错位）的原因。
- 重点分析 `oled_app.c` 及其直接依赖调用链（坐标、字模、缓存、刷新流程）。

## Acceptance Criteria
- [ ] 明确字符翻转问题的触发条件与根因。
- [ ] 明确字符串显示不完整的触发条件与根因。
- [ ] 给出可执行的修复建议（必要时附带修改点）。

## Technical Notes
- 优先按现有驱动和显示流程进行静态排查，不引入额外架构改动。
- 若问题来源不在 `oled_app.c`，需指出上游或下游具体文件与函数。
